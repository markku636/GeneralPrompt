幫我依據過去撰寫程式脈絡撰寫產生指引文件及 Agent Skil,

專門教 AI「怎麼幫你做事」

請以 Claude Code 為例

---

## PROMPT 範例

### 1. 依據新增 command 更新 README 及 CLAUDE.md

```
我新增了一個 command: `xxx`，
請幫我更新 README.md 和 CLAUDE.md，
把這個 command 的用途與用法補上去，格式跟現有的一致。
```

### 2. 依據新增 command 更新 Agent Skill

```
我新增了一個 command: `xxx`，
請幫我更新 agent-skill.md，
新增對應的 skill 描述，格式跟現有的一致。
```

### 3. 依據程式脈絡同步更新指引文件（sync-docs）

> 放在 `.claude/commands/sync-docs.md`，用 `/sync-docs` 觸發

```
掃描當前專案的資料夾結構、技術棧、
.claude/commands/、.claude/rules/、.claude/skills/ 的所有檔案，
比對現有 README.md 和 CLAUDE.md，
把缺少或過時的內容補上，保留現有自訂內容不覆蓋，格式跟現有的一致。
```

### 4. 依據程式脈絡同步更新 Skill（sync-skill）

> 放在 `.claude/commands/sync-skill.md`，用 `/sync-skill` 觸發

```
掃描當前專案的程式碼結構、commands、rules，
比對 .claude/skills/ 底下現有的技能文件，
把缺少的技能補上、過時的描述更新，
保留現有自訂內容不覆蓋，格式跟現有的一致。
```
